# AI-Enabled WordPress Campaigns Landing Page

A professional landing page showcasing RedWagon Agency's AI-powered WordPress campaign solutions.

## Features

- **Modern Design**: Clean, professional layout matching RedWagon Agency branding
- **AI Campaign Focus**: Detailed explanation of AI-enabled WordPress campaigns
- **Responsive Layout**: Optimized for all devices and screen sizes
- **Interactive Elements**: Smooth scrolling, hover effects, and dynamic header
- **Performance Optimized**: Fast loading with minimal dependencies

## Sections

1. **Hero Section**: Compelling headline and call-to-action
2. **Features Grid**: Six key AI campaign features with icons
3. **Benefits Section**: Why choose AI-enabled campaigns
4. **Statistics**: Key performance metrics
5. **Call-to-Action**: Contact section for lead generation
6. **Footer**: Professional closing with branding

## Technology Stack

- Pure HTML5
- CSS3 with modern features (Grid, Flexbox, Gradients)
- Vanilla JavaScript for interactions
- No external dependencies

## Deployment

This is a static HTML page that can be deployed to any web server or hosting platform including:
- Vercel
- Netlify
- GitHub Pages
- Traditional web hosting

The design uses CSS custom properties and modular structure for easy customization of:
- Colors and branding

- Content sections
